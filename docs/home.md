# Welcome to the ClassesCore Wiki.

This documentation covers configuration of ClassesCore, a flexible and scalable class system for Hytale servers.

Whether you’re running a small survival server or a large progression-based network, this wiki will help you configure ClassesCore to fit your needs.

## Getting Started

First, download the mod from [Modtale](https://modtale.net/mod/classescore) or [CurseForge](https://www.curseforge.com/hytale/mods/classescore) and install by following our [Installation Guide](https://wiki.hytalemodding.dev/mod/classescore/installation).

Then start [here](https://wiki.hytalemodding.dev/mod/classescore/configuration) to learn the configuration of ClassesCore.
