This guide explains how to install the mod for both **dedicated servers** and **singleplayer** environments.

## Server Installation

1. **Stop the server** if it is currently running.
2. Navigate to your server’s root directory.
3. Open the `Mods` folder.
    - If the folder does not exist, create one named `Mods`.
4. Place the mod `.jar` file into the `mods` folder.
5. Start the server.
6. Configure the mod using:
    - [Internal Configuration](https://wiki.hytalemodding.dev/mod/classescore/configuration)

## Singleplayer Installation

1. Open the **Hytale Launcher**.
2. Click the **Settings** (⚙️) icon.
3. Under **Directory**, select **Open Directory**.
4. Navigate to the following folder: `Hytale\UserData\Mods`
    - If the `Mods` folder does not exist, create it.
5. Place the mod `.jar` file into the `Mods` folder.
6. Launch Hytale.

## Verifying Installation

- Ensure the mod appears in the in-game mod list when using [Better Modlist](https://www.curseforge.com/hytale/mods/better-modlist)
- Check the game or server logs for any loading errors.
- Confirm configuration files are generated after the first launch.

## Notes

- Always use the **latest compatible version** of the mod for your game.
- Stop the singleplayer world or server before changing configuration values.
